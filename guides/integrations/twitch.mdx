## Twitch User Information

By running the command `twitch [username]`, you can get information on the given profile, such as status, followers, etc.

## Creating a Twitch feed

By adding a feed, bleed will notify members in a channel when a Twitch user starts streaming.

To add a feed from Twitch, run `twitch add #channel [username]`

### Customization Options

<AccordionGroup>
    <Accordion title="Custom Messages">
        Add a custom message that is sent to your channel alongside the notification.
        <Tip>Supports **raw text** and **embed code**! Add `--pingable on` to the command to ping a role in the message.</Tip>
        ```
        // Command syntax
        twitch message [username] (message)

        // Raw text example without ping
        twitch message [username] Bryce is now live!

        // Embed example with ping
        twitch message [username] {embed}$v{description: Bryce is now live!}$v{message:@everyone} --pingable on
        ```
        You can view the message set with `twitch message view [username]`
    </Accordion>
</AccordionGroup>

### Related Commands

<AccordionGroup>
    <Accordion title="Remove a Twitch feed">
        Used to remove a Twitch feed from a channel
        ```
        twitch remove #channel [username]
        ```
    </Accordion>

    <Accordion title="List all Twitch feeds">
        Used to list all Twitch feeds within your server
        ```
        twitch list
        ```
    </Accordion>
</AccordionGroup>
