## About
Setting up the Fortnite integration with Bleed allows you to get automatic shop rotation updates, view information on a Fortnite cosmetic item, set reminders for an item, and more.

<Info>
    You can view the current Fortnite shop rotation by using the `fortniteshop` command.
</Info>


### Viewing information on a cosmetic
To view information on a Fortnite cosmetic, you can use the `fortnite item` command.

```
fortnite item Toosie Slide
```

    <img src="https://i.imgur.com/tcrBSVL.png"/>
</Frame>

### Setting up Fortnite shop notifications
To set up automatic Fortnite shop rotation updates, you can set the channel for the shop updates by using the `fortnite shop` command.

```
fortnite shop #channel
```

<Frame type="glass" caption="Setting where the Fortnite shop updates will be sent">
    <img src="https://i.imgur.com/Zrffkx6.png"/>
</Frame>


### Setting up Fortnite shop voting
To allow your community to upvote or downvote the Fortnite shop, you can enable or disable voting by using the `fortnite shop voting` command.

```
fortnite shop voting true/false
```

<Frame type="glass" caption="Enabling voting for the new Fortnite shop">
    <img src="https://i.imgur.com/VYrYti2.png"/>
</Frame>


### Setting up Fortnite shop pings
To set a role that will be pinged when the Fortnite shop updates, you can set the role by using the `fortnite shop ping` command.

```
fortnite shop ping @role
```

<Frame type="glass" caption="Setting the role for Fortnite shop pings">
    <img src="https://i.imgur.com/vhp6Jzg.png"/>
</Frame>


### Setting up Fortnite item reminders
This is used to be notified when a specific item is in the Fortnite shop. You can set a reminder for an item by using the `fortnite watch` command.

```
fortnite watch Toosie Slide
```

<Frame type="glass" caption="Adding an item to the watch list">
    <img src="https://i.imgur.com/In7xtPf.png"/>
</Frame>

When the item is in the shop, you will receive a notification in your DMs.

<Frame type="glass" caption="Being notified when the item is in the shop">
    <img src="https://i.imgur.com/TsEOPb8.png"/>
</Frame>

To view the items you are watching, you can use the `fortnite watch list` command.

```
fortnite watch list
```
