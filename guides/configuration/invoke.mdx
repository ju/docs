## About

Invoke messages come in two kinds - the message, and the DM. The message is what bleed automatically replies with after a moderation command is ran, which by default is `ğŸ‘`. The DM is what is sent to the user after they are punished, which by default is a simple embed saying the reason and the moderator.

Both the message and the DM can be **raw text** or an **embed**, and all [variables](/extras/variables) can be used.

## Commands

| Command                  | Usage                                               |
| ------------------------ | --------------------------------------------------- |
| `invoke jail dm`         | Changes the DM sent to users on jail.               |
| `invoke jail message`    | Changes the message sent by bleed after punishment. |
| `invoke kick dm`         | Changes the DM sent to users on kick.               |
| `invoke kick message`    | Changes the message sent by bleed after punishment. |
| `invoke tempban dm`      | Changes the DM sent to users on tempban.            |
| `invoke tempban message` | Changes the message sent by bleed after punishment. |
| `invoke softban dm`      | Changes the DM sent to users on softban.            |
| `invoke softban message` | Changes the message sent by bleed after punishment. |
| `invoke warn dm`         | Changes the DM sent to users on warn.               |
| `invoke warn message`    | Changes the message sent by bleed after punishment. |
| `invoke warn dm`         | Changes the DM sent to users on warn.               |
| `invoke warn message`    | Changes the message sent by bleed after punishment. |
| `invoke ban dm`          | Changes the DM sent to users on ban.                |
| `invoke ban message`     | Changes the message sent by bleed after punishment. |
| `invoke hardban dm`      | Changes the DM sent to users on hardban.            |
| `invoke hardban message` | Changes the message sent by bleed after punishment. |
| `invoke timeout dm`      | Changes the DM sent to users on timeout.            |
| `invoke timeout message` | Changes the message sent by bleed after punishment. |
