---
title: 'Command aliases'
description: 'Aliases allow you to add a different trigger for a pre existing command - e.g making ,boot an alternative trigger for ,kick.'
---

## Setup
```
,alias add shortcut command
```
```
,alias add remove ban
```
```
,alias add tunes np
```
