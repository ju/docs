---
title: 'Starboard & Clownboard'
description: 'Starboards and Clownboards are used to have bleed repost messages that have hit a certain amount of a certain reaction to a channel of your choice.'
---
<Info>Both starboard and clownboard use the exact same commands for configuration, to setup the clownboard use `clownboard` in your commands as shown below. </Info>

## Basic setup.

<Steps>
  <Step title="Unlock the starboard.">
    Used to unlock and lock your starboard, this command is used to temporarily block new starred posts. It is automatically set to locked.
    ```
    ,starboard unlock
    ,clowboard unlock
    ```
  </Step>
  <Step title="Set the starboard channel.">
    Set the channel that starred messages will be reposted to.
    ```
    ,starboard set #stars
    ,clownboard set #idiots
    ```
  </Step>
  <Step title="Set the threshold.">
    This is the number of reactions it takes before bleed reposts the message. The minimum threshold is 3.
    ```
    ,starboard threshold 4
    ,clownboard threshold 5
    ```
  </Step>
