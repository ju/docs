## How to add an autoresponder

`,autoresponder add` is the way to get started with autoresponders. This command is used to add an autoresponder to your server - Autoresponders can be either **raw text** or an **embed**.
```
// adding a text autoresponder without parameters
,autoresponder add [trigger], [response]
,autoresponder add welc, welcome to the server!

// adding a text autoresponder with parameters
,autoresponder add welc, welcome to the server! --reply

// adding an embed autoresponder without parameters
,autoresponder add welc, {embed}$v{title:Weclome}$v{description: welcome to the server!}

// adding an embed autoresponder with parameters
,autoresponder add welc, {embed}$v{title:Weclome}$v{description: welcome to the server!} --reply
```
## Parameters
| Parameter used on end of response | Function|
| ------------- | ------ |
`--reply` | Makes bleed use discords reply function on the trigger message.
`--delete` | Makes bleed delete the trigger message after posting the autoresponse.
`--self_destruct 6` | Makes bleed automatically delete the autoresponse after a set amount of time (6 seconds minimum).
`--not_strict` | Makes bleed respond if the trigger message is included anywhere in a sentance, rather than it having to be that exact message.
`--ignore_command_check` | Makes bleed ignore the check for if your trigger message is already a pre existing bleed command - as it wouldn't trigger if this is the case.

<Info>If on mobile, make sure to use `--` as mobile devices autocorrect it to `—` which won't function.</Info>

## Related commands
<AccordionGroup>
  <Accordion title="Restrict by role">
   his command is used to restrict certain autoresponders to certain roles only.

   ```
   ,autoresponder role [role] [allow or deny] [trigger]
   ```
   ```
   ,autoresponder role @staff allow prices
   ```

  </Accordion>

  <Accordion title="Restrict by channel">
    This command is used to restrict certain autoresponders to certain channels only.
   ```
   ,autoresponder channel [channel] [allow or deny] [trigger]
   ```
   ```
   ,autoresponder channel #buy allow prices 
   ```
  </Accordion>

  <Accordion title="Update an autoresponder">
    Use this command to update the response for a pre existing autoresponder.
    ```
   ,autoresponder update [trigger], [new response]
   ```
   ```
   ,autoresponder update welcome, welcome to the server, check out #selfroles
   ```
  </Accordion>
  <Accordion title="Check variables">
    This command returns a link to the [variables](/extras/variables) page.
    ```
   ,autoresponder variables
   ```
  </Accordion>
  <Accordion title="List guild autoresponders">
    Used to list all autoresponses in your guild. 
    ```
   ,autoresponder list
   ```
  </Accordion>
  <Accordion title="Reset autoresponders">
    Used to delete all autoresponses in your guild. 
    ```
   ,autoresponder reset
   ```
   </Accordion>
</AccordionGroup>
