## Setting up the moderation system.
As soon as you invite the bot to a server that it's never been in before, you'll want to set up moderation commands and moderation logs.

> The following commands are used to set up the jail system and all-in-one modlogs, and the in depth mute system.

`,setme`

`,setupmute`
